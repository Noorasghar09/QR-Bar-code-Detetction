About QR Code Scanner

This web application provides a robust and reliable QR code and barcode scanning solution. Our system is powered by machine learning to detect malicious QR codes in real-time and help ensure the safety of users' data.

Technologies Used

  • Python & Flask for backend
  • OpenCV for image processing
  • Machine Learning for maliciousness detection
  • SQLite for report storage
